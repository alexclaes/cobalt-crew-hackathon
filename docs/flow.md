## App Flow – How do users move through the app?

### Happy Path (most important)

### 1. Landing / Home
- Headline: **“Plan a Weekend Trip”**
- Actions:
  - Login
  - Register


### 2. Dashboard
- Button: **“Create new trip”**
- List of previous trips (if any)


### 3. Create Trip
- Trip name (optional)
- Add participants
- Confirm start locations  
  - From user profile  
  - Or manual address input


### 4. Map View
- Start points shown on map
- Calculated midpoint appears
- Suggested cities appear as map pins
- 

### 5. Preferences
- Choose trip focus:
  - Wellness
  - Nature / Active
  - City trip
  - Shopping
- Choose budget:
  - Low
  - Medium
  - High

### 6. Trip Preview
- Destination summary
- Map visualization (start points + destination)
- Categories of suggestions:
  - Hotel
  - Restaurants
  - Bars
  - Activities
- Shareable trip link

### 7. Share & Decide
- Users review the trip via shared link
- Discussion happens outside the app (for MVP)
- _(Later)_:
  - Voting
  - Calendar integration
  - Booking

## TLDR
Login → Create Trip → Add Start Locations → Calculate Midpoint → Choose City → Set Preferences → Preview → Share
